## 架构说明

| 层级 | 技术 | 职责 |
|------|------|------|
| **前端** | Vue 3 + TypeScript + Element Plus | 用户界面、状态管理 |
| **中间件** | Gin Middleware | 认证、CORS、限流、日志 |
| **Handler** | Gin Handlers | HTTP请求处理、参数验证 |
| **Service** | Go Services | 业务逻辑、同步调度 |
| **Platform Clients** | Go HTTP Clients | 各平台API适配 |
| **Repository** | GORM | 数据库操作 |
| **数据库** | PostgreSQL + Redis | 持久化存储、缓存 |

   成本计算说明
商品成本计算方法
加权平均法 (Weighted Average)


新单位成本 = (原库存金额 + 本次入库金额) / (原库存数量 + 本次入库数量)
先进先出法 (FIFO) - 出库时按最早入库的成本计算

标准成本法 (Standard) - 使用预设的标准成本

订单成本计算

订单成本 = 商品成本 + 物流成本 + 包装成本 + 平台佣金 + 服务费 + 推广费用 + 其他费用
毛利 = 实际销售额 - 订单成本
利润率 = 毛利 / 实际销售额 * 100%
